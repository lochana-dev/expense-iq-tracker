# ExpenseIQ Tracker

A modern, intuitive expense tracking web application built with TypeScript. Manage your income, expenses, budgets, and gain valuable financial insights all in one clean, responsive dashboard.

## ✨ Features

- 💰 **Income & Expense Tracking** - Record and categorize all financial transactions
- 📊 **Financial Insights** - Get visual analytics and spending reports
- 💳 **Budget Management** - Set and monitor budget limits for different categories
- 📈 **Dashboard** - Clean, intuitive interface with real-time statistics
- 🎯 **Category Organization** - Organize transactions by custom categories
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- 🔍 **Search & Filter** - Quickly find transactions and analyze spending patterns
